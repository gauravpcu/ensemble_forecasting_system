## üìù Documentation Standards

### Writing Style
- **Clear and concise:** Get to the point quickly
- **Examples included:** Show, don't just tell
- **Actionable:** Provide commands users can run
- **Organized:** Use headers and tables for easy scanning

### Code Blocks
```bash
# Always include comments
python3 script.py  # What this does
```

### Tables
Use tables for:
- Comparing options
- Listing parameters
- Showing metrics
- File references

### Cross-References
- Link between documents
- Use relative links
- Include section anchors
